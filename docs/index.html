Maco Renderer Engine




안녕하세요! 이 문서는 제가 개발한 OpenGL PBR 렌더링 엔진을 소개하고자 작성한 문서입니다.


프로젝트 개요

이 엔진은 물리기반 렌더링을 사용한 엔진이며 제가 학습한 내용을 실제로 구현하며 제대로 학습하였는지 검증을 위한 용도로 제작되었습니다.

이 엔진은 게임엔진이 아닌 렌더링 엔진입니다. 따라서 물리엔진, 씬 관리를 위한 xml기능은 포함되지 않습니다. 그래픽 요소를 표현하고 렌더링 하는걸 목적으로한 작은 프로젝트입니다.

주요 특징

  • 물리 기반 렌더링 : 언리얼 엔진에서 소개된 방식을 이용하여 구현되었습니다. 엔진 내부에서 BRDF LUT을 계산하는게 아닌 IBL 에서 생성한 이미지를 바탕으로 구현됩니다.
  • Bloom : Bloom Disasters를 피하기 위하여 업스케일링과 다운스케일링을 통한 Physically Based Bloom을 사용하였습니다.
  • 그림자 : 더 사실적인 그림자 묘사를 위한 PCF, 그림자 해상도 문제를 해결하기 위하여 CSM을 적용하였습니다.
  • SSAO : Screen Space Ambient Occlusion을 통하여 더 사실적인 렌더링을 하였습니다.
  • 환경 효과 : Screen Space God Rays, Lens Flare Effect를 추가하였습니다.

기술 스택

  • 프로그래밍 언어 : C++
  • 그래픽 API : OpenGL
  • 쉐이더 언어 : GLSL
  • 서드파티 라이브러리: gli, imgui, stbi_image, sdl2, glm
